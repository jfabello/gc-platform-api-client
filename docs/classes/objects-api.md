#### Endpoint

### `postAuthorizationDivisionObject`

Assign a list of objects to a division

#### Description

Set the division of a specified list of objects. The objects must all be of the same type, one of: 
CAMPAIGN, MANAGEMENTUNIT, FLOW, QUEUE, DATATABLES or USER. 
The body of the request is a list of object IDs, which are expected to be 
GUIDs, e.g. ["206ce31f-61ec-40ed-a8b1-be6f06303998","250a754e-f5e4-4f51-800f-a92f09d3bf8c"]

#### Endpoint

`POST /api/v2/authorization/divisions/{divisionId}/objects/{objectType}`

#### Parameters

- `divisionId` - **(string, required)** Division ID
- `objectType` - **(string, required)** The type of the objects. Must be one of the valid object types
- `body` - **(string[], required)** - The body of the request. An empty object or `null` is allowed if the body is optional.
